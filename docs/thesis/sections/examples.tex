\section{Examples}

\subsection{One provider}

Assume that there is only one provider, such that $N_{\mathcal{A}}(1) = \emptyset$. The matching constraint reduces to $X_t = 0 \ \forall t$. This implies that the providers needs to set,

\begin{equation}
    p_t: \ \sum^N_{i = 1} r(s_{i, t}, p_t; \psi_{i, t}) =  M \cdot \left( \E[e_{t+1}] - e_t \right)
\end{equation}
